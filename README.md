# Roads-to-the-TOEIC
만우절 기념 특별 게임잼 기획

2019년 '기대현'씨의 생일 기념 및 4월 1일 만우절을 기념하여 하루만에 뚝-딱 만든 게임 입니다.

- - -

* 방향키를 이용하여 '기대현'을 조작하자!
* SPACE를 누르면 점프! ↓키를 누르면 소닉이 될 수 있다!
* 영단어를 깨우쳐서 토익 점수를 올리자!
* 다른 언어를 공부할 시간이 있습니까 휴면? 영단어가 아니면 당신의 토익 점수가 내려간다구요?

- - -
